## 步驟 1：取得 FinMind API Token（2 分鐘）

### 1.1 註冊帳號

前往 [FinMind 官網](https://finmindtrade.com/)：

1. 點擊右上角「註冊」
2. 填寫 Email、密碼
3. 驗證信箱（檢查收件匣）

### 1.2 取得 API Token

1. 登入後，點擊個人頭像
2. 選擇「API Token」
3. 複製 token（一串英數字）

**範例**：
```
eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

> 💡 **提示**：Free 會員每小時可查詢 600 次，一般使用綽綽有餘！

---

## 步驟 2：設定環境變數（1 分鐘）

### 2.1 設定 Token

#### macOS / Linux

在終端機執行：

```bash
export FINMIND_TOKEN="your_token_here"
```

**永久儲存**（可選）：

```bash
echo 'export FINMIND_TOKEN="your_token_here"' >> ~/.zshrc
source ~/.zshrc
```

#### Windows

在命令提示字元執行：

```cmd
set FINMIND_TOKEN=your_token_here
```

**永久儲存**（可選）：

1. 搜尋「環境變數」
2. 點擊「編輯系統環境變數」
3. 新增 `FINMIND_TOKEN` 變數

### 2.2 驗證設定

```bash
echo $FINMIND_TOKEN
```

應該顯示您的 token。

## 🆘 常見問題

### Q1: 執行時顯示「FINMIND_TOKEN 未設定」

**A**: 
1. 確認步驟 2 是否正確執行
2. 重新開啟終端機
3. 再次執行 `export FINMIND_TOKEN="..."`

### Q2: 顯示「HTTP 402」錯誤

**A**: API 用量超過限制（Free 版 600 次/小時）
- 等待 1 小時後再試
- 或升級 FinMind 會員

### Q3: 找不到股票

**A**: 
- 確認股票代號正確（如 2330）
- 或使用中文名稱（如「台積電」）
- 可查看 [FinMind 股票列表](https://finmindtrade.com/)

### Q4: 回測結果不合理

**A**: 
- 檢查日期範圍是否正確
- 確認策略邏輯符合預期
- 注意：歷史績效不代表未來表現

### Q5: 無法開啟 HTML 圖表

**A**:
1. 確認檔案路徑正確
2. 手動開啟 `reports/` 目錄
3. 找到最新的 `.html` 檔案雙擊開啟

## 💡 使用技巧

### 技巧 1：善用自然語言

**好的描述**：
```
回測台積電的 5 日和 20 日均線交叉策略，過去一年
```

**不好的描述**：
```
2330 MA
```

### 技巧 2：從簡單開始

初次使用建議：
1. 選擇熟悉的股票（如台積電）
2. 使用簡單策略（如均線交叉）
3. 測試短期間（如過去一年）

### 技巧 3：記錄實驗

建立試算表記錄：
- 股票名稱
- 策略類型
- 參數設定
- 回測結果
- 心得筆記

### 技巧 4：理解指標意義

**必懂指標**：
- **總報酬率**：整體獲利百分比
- **最大回撤**：最大虧損幅度
- **勝率**：獲利交易佔比

**進階指標**：
- **Sharpe Ratio**：風險調整後報酬（越高越好）
- **獲利因子**：平均獲利 ÷ 平均虧損（> 1 才合理）

### 技巧 5：不要過度優化

⚠️ **警告**：
- 避免反覆調參數到「最佳結果」
- 這可能導致過度擬合（overfitting）
- 歷史最佳 ≠ 未來最佳

✅ **建議**：
- 使用常見參數組合
- 測試多個時間區間
- 驗證策略邏輯合理性
