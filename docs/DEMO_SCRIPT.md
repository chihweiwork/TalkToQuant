## ❓ 常見問題準備

### Q1: 支援哪些股票？

**A**: 
- 台股全市場（透過 FinMind API）
- 美股（需 FinMind 會員）
- 可查詢股票代號或用中文名稱（如「台積電」）

### Q2: 可以測試自己的策略嗎？

**A**:
- 可以！用中文描述邏輯即可
- 目前支援：均線、RSI、MACD、組合條件
- 複雜策略可手動撰寫 Python 程式碼

### Q3: 歷史資料可以回溯多久？

**A**:
- 依 FinMind 會員等級：
  - Free: 最近 1 年
  - Backer: 最近 3 年
  - Sponsor: 10+ 年
- 資料更新頻率：每日收盤後更新

### Q4: 是否收費？

**A**:
- 專案本身：✅ 完全免費開源
- FinMind API：
  - Free tier：600 次/小時（一般使用足夠）
  - 需要更多：可升級會員

### Q5: 回測結果準確嗎？

**A**:
- 資料來源：FinMind（台灣知名財金資料平台）
- 回測引擎：標準 signal-driven 架構
- 注意事項：
  - ⚠️ 歷史績效不代表未來表現
  - ⚠️ 未考慮交易成本（手續費、稅）
  - ⚠️ 未考慮滑價（slippage）

### Q6: 可以用於實盤交易嗎？

**A**:
- 目前僅支援回測（歷史模擬）
- 不支援實盤交易
- 建議：回測驗證策略 → 手動/API 下單

### Q7: 需要什麼技術背景？

**A**:
- **自然語言介面**：無需程式背景
- **手動撰寫策略**：需要基礎 Python 知識
- **擴充功能**：需要 Pandas、金融知識

### Q8: 如何貢獻或回報問題？

**A**:
- GitHub Issues：回報 bug 或建議功能
- Pull Request：歡迎貢獻程式碼
- Email：chihweiwork@gmail.com
